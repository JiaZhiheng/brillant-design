### Run Locally

To run **Unigine** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/codewithsadee/unigine.git
```

Windows:

```bash
git clone https://github.com/codewithsadee/unigine.git
```
